# Fashion Show Management System

A relational database management system to showcase the  organization and management of fashion shows. The project models key entities such as fashion shows, designers, models, garments, sponsors, organizers, venues, media representatives, and guests using a normalized relational database.

## Features

- Relational database design using MySQL
- Entity-Relationship (ER) modeling
- BCNF normalization
- SQL queries for insights

## Technologies Used

- MySQL
- SQL
- ER Modeling
- Database Normalization (BCNF)

## Project Structure

```

📄 insert_data.sql     → Sample data

```

## Sample Queries

The project includes SQL queries to answer real-world business questions, including:

- Upcoming fashion shows with venue and organizer details
- Sponsors contributing above the average sponsorship amount
- Financially profitable fashion shows
- Multi-table joins and nested subqueries for analytics
